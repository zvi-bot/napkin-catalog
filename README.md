# קטלוג מפיות נייר - ר. שמאי

אתר קטלוג עם 4 דפים (קטלוג, ייצור, מלאי, לקוחות) + פאנל אדמין, מספור סידורי אוטומטי, מצב עריכה מוגן קוד, גרירה ושחרור לסידור תמונות, תפריט לחיצה ימנית ועוד.

## הרצה על שרת / מחשב עצמאי

דורש [Node.js](https://nodejs.org) גרסה 18 ומעלה.

```bash
cd napkin-catalog
npm install
npm start
```

השרת יעלה בכתובת `http://localhost:3000` (או פורט אחר לפי משתנה הסביבה `PORT`).

## סיסמאות ברירת מחדל (לשנות מיד בדף האדמין!)

- **סיסמת עמוד לקוחות:** `1234`
- **קוד עריכה:** `9999`

היכנסו למצב "עריכה" עם הקוד הנ"ל, גשו לדף "אדמין" ועדכנו את שתיהן.

## מבנה הפרויקט

```
napkin-catalog/
├── server.js           # שרת Express + כל ה-API
├── data/db.json         # "מסד הנתונים" - קובץ JSON (דגמים, נושאים, לקוחות, סיסמאות, מספור סידורי)
├── public/
│   ├── index.html
│   ├── styles.css
│   ├── app.js            # כל הלוגיקה בצד לקוח (ניתובים, תצוגה, מצב עריכה, גרירה, תפריטים)
│   ├── assets/            # לוגו "מיוצר בישראל" (שקוף)
│   └── uploads/           # תמונות המפיות שמועלות דרך האתר
```

## מה כבר עובד (גרסה בסיסית - שלב 1)

- 4 דפים + אדמין, ניווט מהתפריט הצדי.
- מספור סידורי אוטומטי (NAP-001, NAP-002...) שלא חוזר לאחר מחיקה.
- מצב תצוגה רגיל: תמונות קבועות, לא ניתנות להזזה.
- מצב עריכה (מוגן קוד): גרירה ושחרור לסידור מחדש, תפריט לחיצה ימנית (העברה בין דפים, החלפת תמונה, שינוי מספר דגם, מחיקה), הוספת תמונה חדשה לקטלוג, הוספת נושא חדש.
- עמוד לקוחות מוגן סיסמה, עם בחירת לקוח מרשימה (וכפתור להוספת לקוח חדש במצב עריכה).
- חיפוש וסינון: לפי נושא בכל הדפים, וחיפוש חופשי לפי מספר דגם בדפי ייצור/מלאי/לקוחות.
- שדה מק"ט הניתן לעריכה ישירה מתחת לכל תמונה (בדפים 2-4).
- דף אדמין לעדכון הסיסמה וקוד העריכה.

## מה נשאר להעמיק (שלבים הבאים)

- ולידציות נוספות והודעות שגיאה מפורטות יותר.
- הרשאות מתקדמות יותר (למשל: תפוגת הרשאת עריכה, היסטוריית שינויים).
- שיפורי עיצוב נוספים, אנימציות עדינות, מצב מובייל מלא.
- גיבוי/ייצוא נתונים.

בהצלחה! אפשר לחזור אליי עם כל שינוי, תוספת או באג שנתקלים בו.
